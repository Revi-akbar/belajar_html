Ringkasan Video 1 – Pendahuluan HTML


Ringkasan Singkat

  • Apa itu HTML? → HyperText Markup Language, bahasa markup untuk menyusun struktur halaman web.
  • Siapa penciptanya? → Tim Berners-Lee, penemu World Wide Web.
  • Sejarah web: dimulai dari kebutuhan berbagi dokumen antar peneliti.
  • Analogi website:
    • Website diibaratkan seperti manusia:
      • HTML → struktur/tulang
      • CSS → gaya/tampilan seperti baju
      • JavaScript → otak/perilaku (interaksi)

Pemahaman Konsep

  • Web dibuat agar bisa mengakses informasi dari mana saja.
  • Browser bertugas membaca kode HTML, CSS, JS dan merender halaman.
  • HTML menyusun elemen-elemen seperti heading, paragraf, gambar, tautan, dsb.

Inti dari Video Ini

  • HTML adalah fondasi utama dalam membuat website
  • Diperlukan pemahaman struktur HTML sebelum mempelajari CSS & JS
  • Video ini belum masuk ke praktik, hanya membangun fondasi pengertian
